# SQL-Based Supplier Performance Analytics Project

This project models a manufacturing supply chain and evaluates supplier reliability using SQL.

## Database Structure
- Suppliers
- Materials
- PurchaseOrders

## Analysis
- Lead Time Calculation
- Delivery Delay Analysis
- On-Time Delivery Percentage
- Average Delay & Risk Identification

## Tools Used
- SQL
- MySQL Workbench
